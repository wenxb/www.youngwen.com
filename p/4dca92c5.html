  • 事务的四大特征：

    一致性 持久性 原子性、隔离性

  • 右外连接查询的是满足条件的数据以及右表所有数据

  • 回滚事务关键字 rollback

  • 创建触发器关键字 create trigger ……

  • delete触发器 的关键字 old 的作用是将要或者已经删除的数据

  • 以下哪个隔离级别是mysql数据库的默认隔离级别 repeatable read

  • 数据库的特征:

    持久化存储数据的。其实数据库就是一个文件系统

    方便存储和管理数据

    使用了统一的方式操作数据库 – SQL

  • 数据表的约束有哪些？

    非空：not null

    唯一：unique

    主键：primary key

    外键：foreign key

  • SQL语句中（添加数据、更改数据、查看数据、删除数据）的语法？
    添加：insert into 表名(列名1,列名2,…列名n) values(值1,值2,…值n);

    删除：delete from 表名 [where 条件]

    更新：update 表名 set 列名1 = 值1, 列名2 = 值2,… [where 条件];

    查看：select 字段名1，字段名2… from 表名 [where 条件] ……；

  • 视图（View）是一种虚拟表存在的表

    简单、安全、数据独立
